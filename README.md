#Introduction
Boilerplate for developing hybrid or desktop app based on JSPM, React & OnsenUI.
The bundled version of the app can then be integrated in either a cordova or an electron prject

#Setup

    npm install
    jspm install

#Workflow

#####Start the development server
    make up

#####Bundle app
    make bundle-sfx
    
#####Copy index    
    make copy-index
    
#####Copy Assets
    make copy-assets
